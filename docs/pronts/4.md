Implementa el módulo de gestión documentaria por obra.

Todos los documentos deben pertenecer a company_id y opcionalmente a work_project_id.

Crear tablas:
- documents
- document_movements
- document_observations
- document_approvals

## Documentos
Campos:
- company_id
- work_project_id
- code
- document_type_id
- subject
- description
- origin_area_id
- destination_area_id
- current_user_id
- status
- priority
- issue_date
- due_date

## Funcionalidades
- Registrar documento
- Adjuntar archivos
- Derivar documento
- Recibir documento
- Observar documento
- Aprobar documento
- Rechazar documento
- Cerrar documento
- Ver timeline
- Ver historial de movimientos

## Estados
- Registrado
- Derivado
- Recibido
- Observado
- En revisión
- Aprobado
- Rechazado
- Cerrado
- Vencido
